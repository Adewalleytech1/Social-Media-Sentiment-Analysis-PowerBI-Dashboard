# Social Media Sentiment Analysis Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze social media sentiment, user engagement, platform activity, and geographic distribution of posts.

The dashboard was developed using a cleaned social media sentiment dataset and provides insights into how users interact across different platforms, countries, and sentiment categories.

The project demonstrates data visualization, dashboard design, KPI reporting, and interactive data exploration using Power BI.

---

## Objectives

The primary objectives of this project were to:

* Import and prepare data in Power BI.
* Create interactive visualizations to analyze sentiment trends.
* Monitor user engagement through likes and retweets.
* Explore platform-wise and country-wise activity.
* Build an interactive dashboard using slicers and filters.
* Generate actionable insights from social media data.

---

## Dataset Information

The dataset contains social media posts with information about:

* Post Text

* Year
* Month
* Day
* Hour

### Dataset Summary

* Total Posts: 732
* Total Likes: 31,000+

## Dashboard Features

### KPI Cards

The dashboard includes key performance indicators:

* Total Posts
* Total Likes
* Total Retweets
* Total Countries

### Visualizations

#### 1. Sentiment Distribution

Displays the frequency of sentiment categories across all social media posts.

#### 2. Posts by Platform

A pie chart showing the distribution of posts across platforms such as:

* Instagram
* Twitter
* Facebook

#### 3. Average Likes by Sentiment

Compares average engagement levels across different sentiment categories.

#### 4. Top Countries by Posts

Highlights the countries generating the highest number of social media posts.

#### 5. Average Likes by Month

Shows engagement trends over time using monthly average likes.

### Interactive Filters

Users can explore the dashboard using slicers for:

* Country
* Platform
* Sentiment

---

## Key Insights

### Sentiment Analysis

* Positive sentiment is the most common sentiment category in the dataset.
* Joy and Excitement are also among the most frequently expressed emotions.

### Platform Analysis

* Instagram contributes the highest share of posts.
* Twitter and Facebook also account for a significant portion of social media activity.

### Geographic Analysis

* The United States records the highest number of posts.
* The United Kingdom and Canada are also among the most active countries.

### Engagement Analysis

* Positive and Joyful sentiments generally receive higher engagement.
* User interactions fluctuate across different months.
